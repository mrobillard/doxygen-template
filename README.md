# doxygen-template

A Doxygen template for an API with a user manual

## Installing Doxygen 

On macOS, you can install Doxygen with Homebrew:

```
brew install doxygen 
```

Otherwise, use the official install [instructions](http://www.doxygen.nl/download.html).
